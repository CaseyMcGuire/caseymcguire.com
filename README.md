This is a starter project for a single-page web app that uses:
- Kotlin
- Spring Boot
- GraphQL
- Relay
- React Router
- React

In order to start:
```
./gradlew bootRun
```

In order to run webpack in development mode:

```
./gradlew webpackDevelopment
```

In order to rebuild relay: 

```
./gradlew buildRelay
```
